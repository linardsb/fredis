# collaborators/

Strategic technical collaborators / network (CRM-style, parallel to `investors/`). For non-investor strategic relationships — technical co-collaborators, sector experts, sounding boards, potential pilot partners.

- `_network.md` — master table (name · domain · status · last contact · next action)
- `{person}.md` — per-person profile (warm/active relationships only — cold stays in the table)

Heartbeat scans `_network.md` daily and surfaces any row where `next action due ≤ today` or any warm contact with > 30 business days since last touch (looser than investors' 10-day threshold — collaborator cadence is slower).
