## Work (`/work`)

### Intro
*three ways saulera engages. read three blocks, pick the one you'd want a call about.*

### Block 1 — agentic workflows that ship
production ai workflows wired into the tools you already use — email, crm, spreadsheets, dashboards, internal docs. built, deployed, and documented so your team runs them without us in the room.

- **typical engagement:** 4–8 weeks, fixed scope.
- **what you get:** the workflow live in your environment, a short runbook, a 30-day support window.
- **best fit:** smbs with a repetitive ops bottleneck — lead triage, invoicing, report generation, content pipelines.

### Block 2 — custom apps and platforms
web and native applications for the workflows that don't fit off-the-shelf. built end-to-end from architecture to deployment.

- **typical engagement:** 8–16 weeks, milestone-based.
- **what you get:** a working product, source code, a handover plan.
- **best fit:** category-specific tooling (mobility, wellbeing, vertical saas) where the right software simply doesn't exist yet.

### Block 3 — martech and email infrastructure
twelve years of email-developer depth at merkle / dentsu. esp architecture, modular component systems, and ai-assisted email workflows that scale across teams.

- **typical engagement:** 2–6 weeks, scoped per audit or build.
- **what you get:** a tightened esp setup, a reusable component library, a workflow that survives team turnover.
- **best fit:** mid-market brands with email programmes that have outgrown their template chaos.

## Results (`/results`)

### Intro
*four agentic systems, built and shipping. each links to the source — dig in.*

### Card 1 — fredis · personal ai advisor and second brain
*built and running since early 2026.*

a personal ai advisor with a 24-skill stack, persistent obsidian-vault memory, and slack / gmail / google workspace / hubspot / github integrations. operates in advisor mode — every output goes to a draft for human review, nothing auto-sends.

- **stack.** python · claude agent sdk · mcp server (fastmcp) · tailscale serve https · obsidian markdown vault · hetzner vps · scheduled cron loops (heartbeat / reflection / synthesis).
- **agentic capabilities.** 24-skill stack with role-based personas · advisor-mode safety boundary (drafts only, no send-authority) · persistent vault memory auto-loaded at session start · heartbeat / reflection / synthesis sdk loops · hooks-based safety enforcement.
- **why it matters.** the flagship proof. a working agentic advisor with persistent memory, integration breadth, and a non-trivial safety model — the exact shape of system saulera builds for clients. *we don't pitch agentic systems. we run on one.*

### Card 2 — email innovation hub · centralised email development platform with ai agents
*github.com/linardsb/merkle-email-hub. built on personal time and stack.*

a centralised email development platform — build, preview, qa, and export production html emails from a single workspace. cms-agnostic, security-first, gdpr-compliant. designed for email teams of 5–50 developers across multiple esps.

- **stack.** fastapi (async python) · next.js 16 / react 19 · postgresql with pgvector · redis · maizzle build pipeline · tailwind / shadcn/ui · docker compose · alembic migrations · vertical slice architecture.
- **agentic capabilities.** nine specialised ai agents (scaffolder, dark mode, content, outlook fixer, accessibility auditor, personalisation, code reviewer, knowledge, innovation) · ai orchestrator routing claude + openai · rag pipeline with pgvector · websocket real-time chat · 10-point qa gate system · zero-trust api (jwt, rbac, row-level security, aes-256, audit trails).
- **why it matters.** production-grade backend, modern frontend, and multi-agent orchestration in one coherent product. direct evidence saulera ships agentic platforms — not just point automations.

### Card 3 — vtv · unified cms and ai agent service for riga city-bus operations
*civic-scale transit operations. source repo to be made public before launch.*

a unified transit operations platform for riga's municipal bus system — a fastapi + pydantic ai agent service paired with a next.js 16 / turborepo cms frontend. gtfs-compliant, swappable llm provider, bilingual (latvian + english), 4-role rbac.

- **stack.** python 3.12 · fastapi 0.120 · pydantic ai · sqlalchemy 2.0 async · postgresql 18 · alembic · mypy + pyright (strict) · ruff · pytest · uv · next.js 16 · react 19 · tailwind v4 · shadcn/ui · auth.js v5 · next-intl · turborepo + pnpm · 75 tests in under 1.2 seconds.
- **agentic capabilities.** single pydantic ai agent with 9 tools — no routing logic, no orchestration overhead · five read-only transit tools (advise, don't decide) · four obsidian-vault tools with `dry_run` previews · swappable llm provider (anthropic, ollama, openai, groq, openrouter) via single env var · openai-compatible streaming chat endpoint · safety constraints baked in.
- **why it matters.** civic and mobility scale, regulated-sector requirements (gtfs, rbac, safety guardrails on tool execution), bilingual delivery — direct evidence saulera ships agentic systems for public-sector and regulated clients, not only commercial smbs.

### Card 4 — ugoki · ai-coached health and wellbeing app (ios + android)
*github.com/linardsb/ugoki-iOS-Android-app. mvp shipped. 2026 re-engagement from walking ventures.*

a mobile wellness app combining intermittent fasting, hiit workouts, and ai coaching for time-constrained professionals. a 15–20 minute daily routine for sustainable health optimisation.

- **stack.** python 3.12 · fastapi · sqlalchemy 2.0 · pydantic 2.0 · pydantic ai · claude 3.5 sonnet / haiku · expo sdk 52 · react native · tamagui · zustand · tanstack query · fly.io · postgresql · cloudflare r2 · expo push.
- **agentic capabilities.** ai coach with constitutional ai guardrails, cross-session memory, skill-based prompt routing · bloodwork analysis (biomarker parsing from uploaded pdfs) · pubmed research hub with ai summaries · health-device sync (apple healthkit + google health connect) · gamification system wired into the ai's coaching context.
- **why it matters.** cross-platform native delivery (ios + android via expo), ai woven into product ux rather than bolted on, plus health-data integrations. evidence saulera ships mobile + ai together — a combination most generalist studios cannot.
